# SentiVoice AI - Project Description

## Overview
**SentiVoice AI** is a cutting-edge sentiment analysis platform that combines high-performance intelligence with a premium, user-centric design. It is designed to analyze emotional depth in both text and speech, providing users with a comprehensive understanding of the underlying sentiment in their communications.

## Key Features
- **Dual-Engine Analysis**: Utilizes a sophisticated hybrid approach combining **Naive Bayes** for categorical classification and **Sequence Logic (LSTM-inspired)** for contextual nuance.
- **Multi-Modal Input**: Supports both traditional text entry and real-time **Voice-to-Text** analysis.
- **Hinglish Intelligence**: Features a specialized lexicon for analyzing mixed Hindi and English (Hinglish), ensuring accurate sentiment detection for regional dialects and slang.
- **Emotional Granularity**: Goes beyond simple positive/negative labels to identify specific emotional states such as *Joyful*, *Anxious*, *Furious*, and *Heartbroken*.
- **Premium Design System**: A state-of-the-art "Red and Black" aesthetic utilizing glassmorphism, smooth animations, and high-contrast visuals for a professional experience.
- **Secure Authentication**: Integrated login and registration system with session-based access control.

## Technical Stack
- **Backend**: Python 3.14+, Flask Web Framework
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Logic**: Custom Mathematical Models (Naive Bayes & Sequence Weighting), Regular Expressions
- **Design**: Google Fonts (Inter, Space Grotesk), Custom CSS Animations

## Target Audience
- **Customer Support**: To gauge client satisfaction and tone.
- **Content Creators**: To understand the emotional impact of their scripts.
- **Developers**: As a foundation for building emotion-aware applications.
